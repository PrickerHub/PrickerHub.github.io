打赢——职场新人求胜指南

杂谈
 2018/02/04

“是故，胜兵先胜而后求战，败兵先战而后求胜。”——《孙子兵法·形篇》

       职场新人，是指刚从大学这个象牙塔里走出来的年轻人。初入职场，从一个舒适区走向不适区，面临生理与心理的双重跃迁，人与人之间往往产生极大的反差，有人平步青云，有人因循守旧，有人不堪重负，有人自我放弃。

       本文以我的所见所闻为引子，讲讲新人如何能够在职场中“打赢”。

心理

担责

       学校不培养心智，学生往往在进入职场数年后才能达到心智成熟。在学校学生们不必关心学习以外的事，“闭门只读圣贤书”，责任心淡薄，往往很难下定决心非干成不可。在职场，做产品，容不得半点懈怠，责任心至关重要，个人的失责，小则任务失败，大则整个产品报废并让他人永远不敢用你。

       责任心是打胜仗的基石，发自内心地担责，迎难而上，即使再小的事，只要允诺过，就负责到底。新人最忌讳推卸责任（俗称甩锅），凡事总以“这不是我的问题”为由而勉强脱身，找借口成了“安身立命”的本能，难有大作为。

       思考如下问题：

  1. 他人都能干好的事我是否有决心干好？——学会担责。

  2. 有哪些活是别人不敢轻易去做的，那是否是我难得的成长机会？——积极担责。

  3. “敢作敢为”带来什么好处？——差异化与不可替代性的构建正是来源于此。

  4. 我今天在工作中有几次以“我是新人”为借口中途放弃或推卸责任？——几次失责。

  5. 我手里的活能否做到事无巨细，负责到底，直到问题有效闭环？——提前识别风险，不要盲目担责.

  6. 我今天在工作中有哪些不称职的行为？哪些事是我本应该负责却没起到关键作用的？我的失职带来了什么后果？我从这些后果中学到了什么？下一次我应该如何避免出现同样的结果？——以结果为导向，完成自我审视。

       推荐阅读：《孤独的幸存者》

抗压

       我认识的20来岁的年轻人中，抗压能力强的屈指可数，大多时常抱怨，以为自己已经承受了很多，实际因为以前丝毫没有担当，所以肩上一旦扛点东西，就会怨声载道。压力不再是内心深处为实现目标而存在的潜在驱动力，而变成了日常晾晒的口舌。

       经常听到身边的同学朋友抱怨工作压力太大，却没有人能说清楚“压力”到底来源于哪里，只要认真仔细深挖压力的根源，压力便能得到有效缓解。

       你打王者荣耀的时候有压力吗？你刷微博在李晓璐博客下面评论“不说了，我去做头发了”的时候有压力吗？你可能会说，当然没有。你有没有仔细想过“为什么没有？”，答案很简单——因为这些事务产生的结果对你的生活毫无影响。

       那么工作中我们的压力来自于哪里？

  1. “接到一个任务，特别难，我感觉压力好大”；

  2. “我手里的活太多了，我一个都搞不定，我该如何应对？”；

  3. “同一天进公司，他为什么升的比我快，我该怎么办？”。

       为什么？一方面来自于人的天性，我们天生对未知的事务充满恐惧与排斥感，另一方面来自于我们对事务结果的过分看待与对不确定性的悲观预期。做不成会怎样？影响整体进度？同事会觉得我能力好差？印象分变低？被贴上“无能”的标签？绩效受到影响，年终奖受到影响？辜负他人的期望？

       我给予如下建议：

  1. 首先，在公司里没人在乎你到底能把事情做成什么样，大家都很忙，不要故步自封，勇于撕掉身上自我构建的标签；

  2. 其次，拥抱磨难，在不断接受的磨难与打击中，抗压力慢慢增强，多给自己一些挑战，如果你每天都过的很舒服，或者只是生理层面的疲惫，那你就要自省了；

  3. 最后，别让“物质”（工资、奖金）成为你的驱动力，这些东西既不牢靠也不持久，发自内心地以不断拓展自我边界为目的来做事，而不是为了拿那么点“钱”来做等量的事，需知钱带来的愉悦感短暂而又浅薄。前三年不要太在意收入，不要动不动就跟周围的人比工资高低，奖金多少，埋头学习干活，三年后再对比也不迟。

       我们的身体在经受创伤自愈后会变得更加坚强。需知没经受过磨难，就难堪大用。“温柔乡是英雄冢”，时刻自省，走出自我舒适区，拥抱苦难。

       推荐阅读：《心胜》

守诺

       小时候学历史，看到古代很多名仕为了兑现承诺不惜牺牲自己的性命，觉得既好笑又不可理喻。直到进入职场，一次次的失信带来巨大的心理落差，让我重新开始思考这个问题。“季布一诺值千金”，为何人们这么重视“守信”？

       “诺”，言若，先出口予之，再如言所行。正常人需经过思考方可行动，有时候我们受荷尔蒙的影响，在不经过大脑反复雕琢的情况下便答应下来远超能力范围的事，往往做出允诺而不守诺之事，从而在他人内心逐渐建立起“轻言寡信”的形象，一旦建立，便难以改变，从而白白失掉很多机会。

       “诺而必达”是领导敢于托付重任于你的前提。我们生活中有太多的正向案例，“连续通宵攻关，只为达成给予客户的最基本承诺——‘业务问题三天内闭环’”、“持续作业36小时抗洪救灾，只为坚持‘以人民为中心’的价值追求——将民众的损失降到最小”、“战后残疾的士兵，妻子不离不弃，只为兑现婚礼当日‘终生陪伴’的诺言”。你说他们傻吗？他们一点也不傻，在履行承诺中，他们承受他人难以想象的艰辛，艰辛带来长足的回报，在别人眼中，他们是“诺而必达”的典范，他们将在今后的工作生活中获得更多的机会，试问谁会将机会留给一个“言语的巨人，行动的矮子”。

       当接受任务之前需思考如下问题：

  1. 我当前能否胜任，如果不能，我能通过自身能力完成百分之多少？——事。

  2. 周围有哪些资源可以支撑我百分之百兑现诺言？——人。

  3. 在时间线上是否有其他关键事件冲突，我能否在deadline之前完成，我该如何排计划？——时间。
    谋定而后动，充分利用身边资源，一旦允诺，一定达成。

       推荐阅读：《把信送给加西亚》

行动

       想要做什么？做成什么样？我该怎么做？往往需要一个明确的目标，需知任何进步均来自于实践，“理论永远是灰色的，实践之树常青”。如果不知道怎么做，当下做好以下几点即可。

做好手头工作

       人为什么工作，被称为“经营之圣”的稻盛和夫先生在《干法》里的解答——“人工作的目的是为了提升自己的心志，只有通过长时间不懈的工作，磨砺了心志，才会具备厚重的人格，在生活中沉稳而不摇摆”。

       经常有同学朋友抱怨“工作毫无意义”、“工作中学不到任何东西”，在下这些结论之前，不妨先扪心自问“何为意义？”、“我想学什么？”。“意义”二字带着主观色彩，不同的人在不同的阶段赋予同一件事不同的“意义”，年轻人当说出“XX毫无意义”的时候，一定要谨慎。

       诚然，工作的意义到底是什么？我的解答是：工作本身并无意义，在完成工作里大大小小的事务中，予事之人不断通过思考与努力攻坚克难，实现自我提升并不断拓展自我界限的的过程，是人赋予工作的“意义”。当然，有人会说，我的工作只是简单的机械重复，毫无“攻坚克难”之说，我相信每一件事都有内在的价值，你觉得轻而易举是你没有想着如何改进，如何把这件事做到极致，如何以自动化代替机械重复。如果我们的祖先永远只做着机械重复的工作而不思改进，人类可能还在茹毛饮血，历史上总有那么些人，从简单地机械劳作中探寻改进的方法从而推动历史进步的巨轮，你为什么不能是这些人？

       当你觉得“前途渺茫”的时候，先做好手头的工作，做到极致。

       推荐阅读：《干法》

不要停止学习

       都知道要终生学习，知易行难。以前总觉得自己读过很多书，前几天跟几个上海的同学聊天，发现他们基本上都是平均一周读一本书，相比之下，两周读一本书的我毫无优势可言。我始终相信书是最廉价但收益颇丰的投资。而且是面向未来的投资，风险为0。不要让人感觉“看起来很努力”。

       如何开始？

  1. 给自己定个小目标，一年读20本书；

  2. 我需要在哪些方面提升个人能力，列一个checklist；

  3. 去论坛里找对应领域的推荐图书，多看评论，看看他人的读后感，列一个2018年书单，也可以每月列一个书单，形式不重要，关键是实践落地；

  4. 按照书单一次只买一本书，看完后再买第二本（这点很重要，达成阶段目标会很有成就感）；

  5. 每天拿出2小时持续投入用来读书（持续投入的意思是找一个安静的环境，远离电脑、手机关机，断绝与外界的一切联系。极端点的做法：建议手机里仅保留微信这么一个社交软件，其他社交娱乐类app全部删除，相信我，没有那么多“重要”的事需要你处理，你消失一年也不会有人在意，我们每天花在刷微博、逛淘宝的碎片化时间太多了，总觉得自己没时间的时候可以审视下时间是否都人为地被碎片化）；

  6. 花10~20分钟写当天的读书笔记（有道云笔记、印象笔记都很好用）；

  7. 坚持4~6，年底作总结。

       推荐阅读：《赢》

定期总结

       理论如果不付诸实践便毫无用处。总结是知识加工的过程，将知识源（书本、网络）中获得的知识原料加工为自己能够消化的一手知识，加工的过程实际上是知识精细化的过程。知识原料在我们的大脑中存在的时间极短。“学而不思则罔”，思考是第一步，也是关键的一步。

与人交流

       工作生活中与人交流是必不可少的环节，如何正确与人交流，我给出如下建议：

  1. 交流问题之前，想清楚交流的目的——我希望为自己实现什么目标？我希望为对方实现什么目标？

  2. 交流问题之前，搞清楚问题的背景、现象、影响、恢复手段、规避措施等。

  3. 如果是想让对方拍板，最好提供选择题而不是论述题。

  4. 考虑交流方的感受——这个问题是否与他利益相关。

  5. 强调共同目的、保持尊重。

  6. 说话要有底气，不清楚的问题不要轻易下结论，搞清楚状况再答复，切忌模棱两可。

       以上只是一小部分，想要深入学习的同学建议阅读《非暴力沟通》、《关键对话》这两本书。

       推荐阅读：《关键对话》

最后

       打赢人生这场恶仗，从职场开始。
