《跃迁》

读书笔记
 2017/10/27

       这本书有点….处处都是“道”，不过有一些新颖的观点，可以一看。

高手的暗箱

  • 信息大爆炸时代特征
  1. 信息变多、思考变浅；
  2. 机会变多、竞争跨界；
  3. 随时干扰、永远在线。
  • 三种时代溺水者
  1. 无法掌握注意力；
  2. 无法过滤信息，找不到重点；
  3. 不理解系统，无法与陌生人联机协作。

       反之，管理注意力、判断信息、和陌生人社会协作，掌握了这些“时代魔法”，你会成为真正的“超人”。

  • 底层规律

       成功不仅限于努力、天赋、机遇，利用底层规律，激发个体跨越式成长。所谓底层规律是指行业的“隐秘知识”，也就是套路，就是门道。

高手战略

  • 站位比努力更重要

       “二八法则”、“马太效应”、“长尾理论”——极少数人享有绝大多数的资源（富人越来越富有，穷人越来越穷）

  • 两个规律杠杆

       对内，通过二八法则的三次方，持续放大自我效能；

       对外，通过移动到系统的头部，获得系统巨大的推动力。

  • 持续放大自我效能

       做事情一定要抓重点。社会和自然大部分系统都有重点。

       在非重点事务上“得过且过”。

  • 怎样找到头部

高价值：投入产出比最高的20%的赛场。

高优势：实力排在赛场序列的前20%。

  • 头部效应的两大误区

误区一：从当前优势出发。

  1. 是不是优势，需要入场才知道
  2. 过去的优势不等于未来的优势

误区二：着急入场，不想优势。

       看见高价值区就撸起袖子开始做，很少思考和别人相比，自己有什么优势。

联机学习

  • 为什么大多数人认知效率低下？
  1. 一手信息：知识的源头；
  2. 二手信息：忠实转述一手信息；
  3. 三手信息：为传播而简化和极端化观点的陈述；
  4. 四手信息：出于各种动机充满个人经验的情绪化表达。

       我们获取的知识绝大多数是N手信息，这是认知低效的主要原因。

       想想微博、微信里我们转发的东西，是几手信息？

  • 如何提高认知效率？

第一：识别知识的源头——一手信息。

第二：功利读书法

  1. 极其功利地少读书
  2. 极其功利地配置资源
  3. 不要从第一页开始读书
  • 如何在需要的时候，调取知识？

       这块其实就是“总结、总结、总结”，将获得的知识结构化（因果、关联）

  • 学习的三种境界
  1. 自学：自己找答案
  2. 联机学习：和同行交流，答案换答案（找书、网络）
  3. 跨界联机学习：跨行学习交流，用答案换答案（找高手建立连接）
  • 知识IPO

输入问题、解决问题、输出产品。

我觉得就是：遇到问题→解决→总结

  • 构建自己的知识体系

       站在知识源头，萃取知识晶体（也就是自我学习+总结）；

       联机学习，用一块晶体换来更多晶体（与高手建立连接）；

       以问题为中心学习，创造自己的问题树；（自我知识体系结构）

       用知识IPO让知识变成价值。

破局思维

  • 新手看树木，高手看森林

系统看世界：

元素、关系和功能。关系很重要

  • 生活中的无解问题，真的无法解决吗？

无法改变“元素”，就去改变“关系”

  • 看问题的方式
  1. 看得远：能看到事物发展的脉络，找到过去和现在的关系，找得到“回路”；
  2. 看得透：能够理解事情背后的真正规律，看到事情背后的“层级”。
  • 回路：人生最常见的8种循环

       4种增长引擎：

  1. 好习惯
  2. 从兴趣培养到能力养成
  3. 快速学习的知识IPO
  4. 企业管理中的“信-任”循环

       4种死亡螺旋：

  1. 穷者越穷
  2. 投入不足
  3. 工作狂循环
  4. 做自己（求认同-找不到自己-求认同）
  • 层级：看问题很透彻的技术

现象→规律→心智→价值观

内在修炼

  • 看世界：开放而专注
  1. 见得人好，经常随喜赞叹。
  2. 找到自己领域的知识源头，并分享。
  3. 不随便崇拜谁。
  4. 不再认为自己不喜欢、看不懂的东西就是傻的。
  5. 小心那些“一切都能解释得通”的上帝视角感理论。
  6. 对水平没你高的人要宽容，因为你也没有掌握真理。
  7. 留出10%~30%的时间，给自己不懂也不太会接触的领域。
  8. 站在更高角度，发现和专注于自己的独特之处。
  9. 专注于自己的人生大问题。
  • 看自己：迟钝而有趣
  1. 对不重要的事，漠不关心。
  2. 忍住第一反应，等待第二反应。
  3. 不追热点，等要点浮现。
  4. 寻求整体最优解，站在长周期做判断。
  5. 多元，定期做点儿不靠谱、有趣无用的事。
  6. 成功是小概率事件，找到自己的无限游戏。
  7. 放下焦虑，不要放下好奇心。
  • 看人际：简单善良可激怒

“或曰：‘以德报怨何如？’子曰：‘何以报德？以直报怨，以德报德。’” ——《论语·宪问》

总结

  1. 高手战略：超越个体努力，借助社会杠杆成长；
  2. 知识IPO法则：停止单机式学习，成为联机式学习者；
  3. 系统思维：跳出平面思维，创造性解决问题；
  4. 内在修炼：摆脱农业思维，在当今时代成为幸福的高手。
